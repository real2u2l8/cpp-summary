# CPP Toy Projects

이 폴더는 C++ 실습 프로젝트들을 포함하고 있습니다.

## 프로젝트 구조
각 프로젝트는 독립적인 폴더로 구성되어 있으며, 다음과 같은 구조를 따릅니다:
- 소스 코드 파일들
- 프로젝트별 README.md (상세 설명 포함)
- Visual Studio 프로젝트 파일 (Windows)
- Makefile (Linux)

## 실행 환경
### Windows
- Visual Studio 2019 이상
- C++17 이상

### Linux
- GCC 7.0 이상
- CMake 3.10 이상
- C++17 이상

## 학습 목표
- 실제 프로젝트를 통한 C++ 실무 능력 향상
- 크로스 플랫폼 프로그래밍 경험
- Windows/Linux 시스템 프로그래밍
- 다양한 프로그래밍 패턴 및 기법 학습
